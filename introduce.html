What is Game of Life ?

Game of Life is a game invented by a mathmatician John Horton Conway

It starts with some life cells spread on the canvas, and they evolve accroding to some rules.

Rule

(1) Any live cell with fewer than two live neighbours dies, as if caused by under-population.

(2) Any live cell with two or three live neighbours lives on to the next generation.

(3) Any live cell with more than three live neighbours dies, as if by overcrowding.

(4) Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Special Patterns

If strated with some specific patterns, you will get some interesting results.

Still Life

If you start with these patterns, the cells will be alive forever:

*Block *Beehive *Loaf *Boat

Oscillators

If you start with these patterns, the cells will change but come back to the original pattern you start.

*Blinker *Toad *Beacon *Pulsar

Spaceships

If you start with these patterns, the cells are never stable, but the pattern shifts.

*Glider *LWSS

How is our game of life different?

More funs !!!

We create some new features including infected cells, mines, pacman and snakes.

Infected Cells

Infected cell is like a regular ceil but of different colors. It cannot appear on its own and will only be generated by the user. If a life cell have at least on infected cell neighbour, then it becomes infected as well. The exception case is that a infected cell has exactly three infected neighbour, then it becomes normal again.

Snake

Snake is generated randomly. All the cells will be dead after it pass them.

If a snake pass a mine, the mine will explode as well.

If the snake touches its head, it will stop and become dead.

Pacman

Pacman will eat all the mines and kill all the cells when it pass by.

Mines

The mines are generated randommly. If a life cell or snack touch the mine, it will explode and kill a range of its neighbour, (so you will see a circle of red dead cells)
